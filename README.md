## Descrição 
O site é um sistema de catálogo desenvolvido para servir como uma vitrine digital para uma empresa de mármore, permitindo que os clientes encontrem os serviços prestados, além das peças e pedras trabalhadas.
O projeto foi criado com o intuito de profissionalizar ainda mais a empresa e oferecer uma forma organizada e visualmente clara para os clientes conhecerem tudo o que é oferecido.

Ele foi construído usando PHP puro no BackEnd, com apoio de ferramentas como Composer e Autoload para agilizar o processo de desenvolvimento.
No FrontEnd, foram utilizados HTML, CSS, Bootstrap e JavaScript, tudo sem frameworks adicionais.
